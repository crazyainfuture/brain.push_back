# XT_Journey

這個倉庫用來整理我的大學學習歷程，內容包含課程筆記、作業、實驗紀錄、修課心得，以及部分程式實作。

## 倉庫內容

### 修課心得
- 各課程的修課經驗與整理心得。

## 這裡會放什麼

- 課堂筆記與重點整理
- 作業、程式碼與實驗紀錄
- 修課心得與學習歷程備忘
- 不同學期課程的分類歸檔


## 使用方式

如果你是來找課程資料，可以依照年級、學期與課名進入對應資料夾。
- note 是課堂筆記
- hw 作業 (供參考)

如果你是來看程式碼，建議優先從以下路徑開始：
- `大一(智慧資安)/JAVA_OOP/java_code/`
- `大二上/組語/asm_code/`
