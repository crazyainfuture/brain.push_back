# XT_Journey

這個倉庫用來整理我的大學學習歷程，內容包含課程筆記、作業、實驗紀錄、修課心得，以及部分程式實作。

## 倉庫內容

### 修課心得
- 各課程的修課經驗與整理心得。

### 大一（智慧資安）
- `CAL2`
- `JAVA_OOP`
    - 課堂講義 PDF
    - `java_code/`：Java 練習、作業與 package/OOP 範例
- `人工智慧概論`
- `數位邏輯設計`
- `資訊安全概論`
- `離散課堂筆記`
    - `note.md`：離散數學課堂筆記

### 大二上
- `數導`
- `組語`
    - `asm_code/`：組合語言作業與練習
    - 另含多份作業 PDF
- `線代`
- `軟工`
    - `location.md`：課程相關位置或記錄

### 大二下
- `數位邏輯實驗`
    - `lab02.bdf`
    - `Readme.md`
- `計組`

## 這裡會放什麼

- 課堂筆記與重點整理
- 作業、程式碼與實驗紀錄
- 修課心得與學習歷程備忘
- 不同學期課程的分類歸檔

## 使用方式

如果你是來找課程資料，可以依照年級、學期與課名進入對應資料夾。

如果你是來看程式碼，建議優先從以下路徑開始：
- `大一(智慧資安)/JAVA_OOP/java_code/`
- `大二上/組語/asm_code/`

## 備註

這些內容主要是我個人的學習整理，可能有錯誤、遺漏，或不一定適合每一門課與每一位同學。請把它當作參考資料，而不是標準答案。
