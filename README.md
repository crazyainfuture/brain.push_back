# XT_Journey

這個倉庫用來整理我的大學學習歷程，包含課程筆記、作業、實驗紀錄、修課心得，以及部分程式實作。

## 倉庫內容

### 1. 修課心得
- 各課程的修課經驗與整理心得。

### 2. 大一（智慧資安）
- `CAL2`
- `JAVA_OOP`
- `人工智慧概論`
- `數位邏輯設計`
- `資訊安全概論`
- `離散課堂筆記`

### 3. 大二上
- `數導`
- `組語`
    - 含 `asm_code` 組合語言程式練習與作業
- `線代`
- `軟工`

### 4. 大二下
- `數位邏輯實驗`
- `計組`

## 這裡會放什麼

- 課堂筆記與重點整理
- 作業、程式碼與實驗紀錄

如果你是來找課程資料，可以直接依照年級、學期與課名進入對應資料夾。
